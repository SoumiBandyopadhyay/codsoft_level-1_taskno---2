# 🌍 TravelX - Responsive Travel Landing Page

A modern and fully responsive Travel Landing Page built using HTML, CSS, and JavaScript.

## 🚀 Project Overview

TravelX is a beautiful travel website landing page designed to showcase travel services and destinations. The website features an attractive hero section with a travel-themed background image, feature cards, and a responsive layout that works seamlessly across desktop, tablet, and mobile devices.

## ✨ Features

- Responsive Design
- Full-Screen Travel Background Image
- Modern Hero Section
- Interactive Call-to-Action Button
- Feature Cards with Hover Effects
- Mobile-Friendly Layout
- Clean and Professional UI
- Footer with Author Information

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
